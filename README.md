GO
===============


## Go描述

* 一套基于**Gson+Ormlite**的企业级快速开发框架,简写GO.主要包含2部分.


> * 使用开源OrmLite框架进行ORM本地数据库操作:[详细使用,请猛击我吧](http://ormlite.com/ "OrmLite")
> * 使用Google官方的**Gson**工具来映射json与bean.
